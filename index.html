Projects

Railway Booking

As part of my coursework, I developed a railway booking system in C. This project aimed to create a user-friendly interface that allows users to book train tickets, check seat availability, and view train schedules. The system efficiently handles booking operations, updates seat availability in real-time, and provides a summary of booked tickets. Through this project, I honed my skills in data structures, file handling, and algorithm optimization, reinforcing my understanding of the C programming language and its practical applications.

Rock Paper Scissors

I developed a Rock-Paper-Scissors game in Python as a fun and interactive way to apply my programming skills. This project features a simple command-line interface where users can play the classic game against the computer. The game uses random number generation to simulate the computer's choices and includes logic to determine the winner based on the standard rules of Rock-Paper-Scissors. Through this project, I enhanced my understanding of Python basics, including loops, conditionals, and user input handling, while also creating an enjoyable user experience.

Sudoku Game

I developed a Sudoku game in C++ as part of my programming projects. This game allows users to play Sudoku with varying levels of difficulty, providing a complete and interactive puzzle-solving experience. The game includes features such as generating new puzzles, validating user input, and checking for puzzle completion. Through this project, I deepened my understanding of object-oriented programming principles, data structures, and algorithms in C++. It also enhanced my problem-solving skills and my ability to create engaging user interfaces.
